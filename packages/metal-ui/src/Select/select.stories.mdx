# Select

Use the `options` prop to set select options.
To provide a placeholder, add an option with value "" and disabled true.

```json
[
  {
    "value": "",
    "name": "Choose One",
    "disabled": true
  },
  {
    "value": "option_1",
    "name": "Option 1"
  },
  {
    "value": "option_2",
    "name": "Option 2"
  },
  {
    "value": "option_3",
    "name": "Option 3"
  }
]
```
