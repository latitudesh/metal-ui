# Available components

Metal exports the following components for Dropdowns:

- Dropdown
- DropdownTrigger
- DropdownContent
- DropdownLabel
- DropdownItem
- DropdownSeparator

# API

This component uses the Radix UI `DropdownMenu` primitive. Refer to [this page](https://radix-ui.com/primitives/docs/components/dropdown-menu#api-reference) for the API reference, but note components in Metal UI do not use dot-notation.

# Custom Trigger

When using a `button` as a custom trigger, set `DropdownTrigger` as a different element to avoid nesting buttons.

<Preview>
  <Story name="Custom Trigger">
    <Dropdown>
      <DropdownTrigger asChild="span">
        <Button label="Trigger" />
      </DropdownTrigger>
      <DropdownContent>
        <DropdownItem onSelect={() => console.log("cut")}>Cut</DropdownItem>
        <DropdownItem onSelect={() => console.log("copy")}>Copy</DropdownItem>
        <DropdownItem onSelect={() => console.log("paste")}>Paste</DropdownItem>
      </DropdownContent>
    </Dropdown>
  </Story>
</Preview>
