# With Icon

The component does not handle icon sizing. You should manage that from your project. `20px` is usually the recommended size.

<Preview>
  <Story name="With Icon">
    <Box flex justifyContent="around">

        iconAfter={<ArrowUpOutline size={20} />}

        iconBefore={<ArrowUpOutline size={20} />}
