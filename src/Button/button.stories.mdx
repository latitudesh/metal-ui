# With Icon

The component does not handle icon sizing. You should manage that from your project. `20px` is usually the recommended size.

<Preview>
  <Story name="With Icon">
    <Box flex justifyContent="around">
      <Button iconBefore={<ArrowUpOutline size={20} />} label="Icon before" />
      <Button
        iconAfter={<ArrowUpOutline size={20} />}
        variant="secondary"
        label="Icon after"
      />
      <Button
        iconBefore={<ArrowUpOutline size={20} />}
        iconAfter={<ArrowUpOutline size={20} />}
        variant="secondary"
        label="Icon before and after"

    </Box>
  </Story>
</Preview>

# With `next/link`

For use in conjunction with NextJS' `next/link`, wrap the button with `<Link>` and set `passHref` to true.

```javascript
<Link href="https://www.maxihost.com" passHref>
  <Button
    label="Go to website"
  />
</Link>
```
