# Provide Email

Email can be provided via a prop in which case the user does not have to enter it.

<Preview>
    <Story name="Provide Email">
        <div className="flex h-full items-end justify-center w-full my-10" style={{ height: '300px' }}>
            <Feedback url="https://httpbin.org/post" email={"test@email.com"} />
        </div>
    </Story>
</Preview>

# Custom Trigger

<Preview>
    <Story name="Custom Trigger">
        <div className="flex h-full items-end justify-center w-full my-10" style={{ height: '300px' }} >
            <Feedback url="https://httpbin.org/post" side="top">
                {({ open, setOpen, ref }) => <div style={{ width: '300px' }}>
                <Button className="absolute" size="normal" ref={ref} onClick={(e) => {
                    e.preventDefault()
                    setOpen(!open);
                }} label={"Custom Trigger"}/>
                </div>}
            </Feedback>
        </div>
    </Story>
</Preview>
